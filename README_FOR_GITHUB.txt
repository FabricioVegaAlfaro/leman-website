
LEMAN website - listo para GitHub Pages con dominio www.leman.guru
-------------------------------------------------------------------

Este paquete contiene:
- index.html
- CNAME (www.leman.guru)
- .nojekyll

Pasos básicos:
1. Crear un repositorio en GitHub, por ejemplo: leman-website
2. Subir estos archivos (sin el ZIP) al repositorio
3. En Settings → Pages → Source → main branch / root
4. En Custom Domain: www.leman.guru
5. En Squarespace Domains:
   - CNAME: Host: www → Value: <tuusuario>.github.io
   - A records: (para el dominio raíz opcionalmente)
     185.199.108.153
     185.199.109.153
     185.199.110.153
     185.199.111.153
6. Guarda los cambios y espera la propagación.
